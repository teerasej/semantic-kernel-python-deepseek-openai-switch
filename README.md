# Project Setup

## Create the `.env` file

Create a `.env` file in the root directory of your project with the following content:

```
OPENAI_API_KEY=
OPENAI_MODEL=gpt-3.5-turbo
OPENAI_ORGANIZATION=
OPENAI_URI=

DEEPSEEK_API_KEY=
DEEPSEEK_MODEL=deepseek-chat
DEEPSEEK_URI=https://api.deepseek.com

API_KEY=
MODEL_ID=gpt-3.5-turbo
SERVICE_URI=
```
